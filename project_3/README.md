### Project Description
- Creating a model based on machine learning algorithms that predicts hotel ratings.

### What case are we solving?
- One of Booking's problems is dishonest hotels that inflate their ratings. If the predictions of the created model differ significantly from the actual results, then
the hotel may be acting dishonestly and should be investigated.

**Competition Terms:**
- Participate in the competition on Kaggle.com
- Improve the accuracy of the model's predictions by more thoroughly preparing the data.

**Quality Metric**
- MAPE - Mean Absolute Percentage Error
- MAE - Mean Absolute Error

**What We Practice**
- Learn to apply data analysis and feature engineering to intelligence applications.

### Data Summary
- hotels_train.csv - training dataset
- hotels_test.csv - quality assessment dataset
- submission.csv - submission file in the required format
- base solution as a basis
- Notebook on Kaggle.com: https://www.kaggle.com/code/aleksejszudiins/project-3-eda-feature-engineering
